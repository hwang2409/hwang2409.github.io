# Technical Easter Egg Ideas

- Semantic backlinks: build-time embeddings connect related paragraphs across posts, exposed through tiny text affordances.
- Model surprisal view: `normal / model` mode shades words by model surprisal while staying grayscale/static.
- Post graph page: generated plain-text adjacency list between posts, concepts, and related sections.
- Build stats page/footer: post, paragraph, code block, syntax-token, commit, and timestamp metadata.
- Identifier index: generated index of code identifiers across posts, extending the token inspector.
- Definition memory: later appearances of technical terms can point back to first introduction.
- Static search with debug mode: search results can expose simple scoring terms and matches.
